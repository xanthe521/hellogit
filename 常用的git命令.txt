$ git init
    创建本地git仓库,创建后的默认分支名为master
    
$ ls
    查看当前目录下的文件
    
$ ls -af
    查看当前目录下的文件(如果存在.git文件也会被显示)
    
$ git branch
    查看仓库所有的分支,当前分支名前有*
    
$ git help
    查看git帮助
    
    
$ git add .
    将文件添加到暂存区,  .  表示添加全部文件
    
$ git status -s 
    查看暂存区中的文件
    
$ git commit -m 'XXXXXX'
    向本地仓库提交,本次提交的注释为'XXXXXX'
    
$ git push <alias> <branch>
    向远程仓库<alias>提交分支<branch>
